OuRecipe
===================

----------
### Install Backend
- `$ npm install`
- setup config => /Server/config/config.js
- setup database => /Server/config/database.js
- `$ npm run-script db:migrate`
- `$ npm run-script db:seed`
- `$ npm run-script run`

----------
### Install Frontend
- `$ npm install`
- `$ bower install`
- run & watch : `$ npm run-script run`
- build production / minify: `$ npm run-script built`
